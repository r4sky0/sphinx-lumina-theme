# Keyboard shortcuts

Lumina is built to be used without a mouse. Press {kbd}`?` on any page to pull up the same reference as this one.

## Search

| Shortcut | Action |
| --- | --- |
| {kbd}`⌘K` / {kbd}`Ctrl+K` | Open the search modal |
| {kbd}`/` | Open the search modal (alias) |
| {kbd}`↑` / {kbd}`↓` | Move between results |
| {kbd}`↵` | Open the selected result |
| {kbd}`Esc` | Close the search modal |

The `/` alias is suppressed while the user is typing in an input, textarea, select, or contenteditable element, so it stays out of the way when you're filling out a form.

## Navigation

| Shortcut | Action |
| --- | --- |
| {kbd}`Esc` | Close any open dialog (search, shortcuts, sidebar drawer) |
| {kbd}`Tab` / {kbd}`Shift+Tab` | Move focus; focus is trapped inside open dialogs |
